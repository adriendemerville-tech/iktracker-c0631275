# IKtracker : Assistant Intelligent d'Indemnités Kilométriques

## Description du Produit

IKtracker est une application SaaS gratuite pour automatiser le calcul des indemnités kilométriques (IK) via la synchronisation d'agenda (Google/Outlook).

## Proposition de Valeur Unique (USP)

- **100% Gratuit** : Pas de frais cachés, développé par Adrien de Volontat (Avenir Rénovations).
- **Automatique** : Détection des adresses dans l'agenda et calcul via API Google Maps.
- **Conforme 2026** : Barème fiscal français officiel à jour.

## Fonctionnalités Techniques

- Synchronisation Google Calendar / Outlook
- Gestion Multi-véhicules (Voiture/Moto) et Puissance Fiscale
- Mode Tournée pour Infirmiers/Livreurs
- Export PDF/CSV conforme URSSAF

## Cibles

Indépendants, Libéraux (Santé, Droit), Artisans, Commerciaux.
